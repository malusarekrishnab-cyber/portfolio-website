# Krishna Malusare — Portfolio

Personal portfolio website built with React, Vite, and Tailwind CSS.

## Tech Stack

- React + Vite
- Tailwind CSS + shadcn/ui
- Framer Motion (animations)
- EmailJS (contact form)

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

Deployed on [Vercel](https://vercel.com).
